# Healther — UX/UI Итерация 3: Информационный каркас

> Получено: 05.07.2026 от Саши
> Статус: ТЗ, ожидает реализации

## 1. Глобальная навигация (Bottom Tab Bar)
5 вкладок:
1. **Главная (Dashboard)** — сводка на сегодня, алерты, расписание, остатки лекарств
2. **Аналитика (Labs & Scans)** — анализы, УЗИ/МРТ, дневник симптомов
3. **Аптечка (Meds)** — активные лекарства, история, рецепты
4. **Медкарта (Records)** — диагнозы, история визитов, флюорография
5. **Профиль (Profile)** — ФИО, антропометрия, ИМТ, страховка, настройки

## 2. Dashboard
- **Блок 1: Алерты** — RED/YELLOW dismissible-баннеры. Если нет — скрыт.
- **Блок 2: Ближайший визит** — 1-2 pending визита, крупная карточка.
- **Блок 3: Инвентаризация аптечки** — горизонтальная карусель мини-карточек с прогресс-баром остатков.
- **Блок 4: Дневник** — Quick Action: "+ Давление", "+ Пульс" + последние значения за 3 дня.

## 3. Cross-linking (мягкие связи)
- Bottom Sheets для деталей препарата/диагноза
- Теги-ссылки: [Врач: ...], [Диагноз: ...]
- При тапе — встроенный поиск по базе

## 4. Оптимизация списков
- ❌ Без серверной пагинации (данных < 200 строк)
- Анализы: группировка по эпизодам + Segmented Control по годам
- Дневник симптомов: группировка по месяцам
- Поиск: Sticky Header, мгновенная фильтрация на клиенте

## 5. Empty States
- Серая иконка + "Здесь пока ничего нет" + кнопка действия
